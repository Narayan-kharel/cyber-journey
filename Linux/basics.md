# Linux Basics Notes

## Command: `pwd`
- Shows the current directory (path) you're in.
- Example output: `/home/kali`

## Command: `ls`
- Lists files and directories in the current directory.

## Command: `ls -a`
- Lists all files, including hidden ones (those that start with a dot).

## Command: `ls -l`
- Lists files with detailed information (permissions, size, modification time).

## Command: `clear`
- Clears the terminal screen.
